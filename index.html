Easy OTP is only for quick browser use. Nothing is uploaded, synced, or sent anywhere.

Important: Easy OTP is only meant for quick browser use and temporary access to TOTP codes.
Saved keys stay only in your browser using local storage and may disappear if you clear cache, cookies, browser data, or use a cleaner tool.
For important accounts, it is safer to use dedicated apps such as Google Authenticator, Microsoft Authenticator, Authy, or similar authenticator apps.
